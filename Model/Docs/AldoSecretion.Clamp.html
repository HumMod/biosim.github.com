AldoSecretion.Clamp

Definition

The clamp feature of HumMod enables the model user to fix
aldosterone secretion at a particular level. If the clamp
is off, then aldosterone secretion responds normally to
the various physiologic stimuli which affect it.
If the clamp is on, then aldosterone secretion is set
by the user and A2 Effect and K+ Effect are both 1.0
