Pheochromocytoma.InEpisode

Definition

The model is set up with an intermittently secreting tumor.
The tumor secretes catecholamines several times a day.
InEpisode refers to whether the tumor is actually secreting
catecholamines or not. If InEpisode is false, then the
tumor is non-secreting at that particular time. If it is true
then the tumor is secreting.
