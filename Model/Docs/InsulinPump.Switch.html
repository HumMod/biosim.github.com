InsulinPump.Switch

Definition

When the insulin pump switch is turned on, the model
user can set infusion rates using the sliding bar
feature in the insulin pump display box.

Physical Units

Units for insulin pool are mU.
Units for [plasma] are uU/mL.

uU * 0.04167 = nG
uU * 0.007241 = pMol


An international unit (U) is 1/24 mG.
Molecular weight is 5754.
