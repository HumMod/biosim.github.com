EPOSecretion.Base

Definition

Base secretion rate is the initial starting secretion rate
of EPO. Base values can be set by the model user.

Overview

EPO, the hormone that stimulates red blood cell
production, is synthesized in the kidney and liver
in response to hypoxia. Studies in isolated perfused
kidneys suggest that it is tissue oxygenation,
and not arterial PO2 which regulates EPO production
in the kidney. EPO secretion depends on kidney
function. Chronic renal disease can lead
to anemia because of a reduction in EPO secretion.
