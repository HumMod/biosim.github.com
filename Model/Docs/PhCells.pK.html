PhCells.pK

Definition

For any acid, the concentration of the acid relative to its
dissociated ions is defined by the dissociation constant K.

pK = -log K

Concentration Units

pK is a constant with value of 7.42
pK is calculated from the following equation:

pH = pK + log10 (SID/pCO2) and using the following data:
pH = 7.37
SID = 40mEq/L
pCO2 = 45 mmHg
