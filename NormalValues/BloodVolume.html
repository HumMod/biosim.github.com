Normal Blood Volume - Males

Reference Man uses a value of 69.3 mL/kG body weight for estimating blood volume. Yiengst and Shock reported a value of 75.0. The average is 72.1 mL/kG.

Body Mass (kG) Blood (mL/kG) BV (ml)
75 72.1 5408

Normal Blood Volume - Females

Reference Man uses a value of 64.4 mL/kG body weight for estimating blood volume. Wadsworth reported a value of 66.5. The average is 65.5 mL/kG.

Blood Volume - Effect of Body Composition

This is a preliminary report on scaling blood volume values. The general idea in scaling is that increased amounts of adipose tissue are associated with rather modest increases in blood volume.

I'm shooting for an algorithm similar to the one used with scaling cardiac output, where the calculated value is a function of body size and composition.
