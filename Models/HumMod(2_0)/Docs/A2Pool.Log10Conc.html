A2Pool.Log10Conc

Definition

Log10Conc converts [A2(pG/mL)] to a logarithmic
value. If [A2(pG/mL)] is less than 1.0, then Log10Conc
has a value of zero.

XXX

[Plasma] pG/mL
Plasma Mass nG
Fluxes nG/Min
Stored Mass uG

Notice that micro (10^-6), nano (10^-9) and pico (10^-12)
gram units are used.
