EpiPool.Gain

Definition

Gain is defined as positive changes in mass of epinephrine
and is a function of epinephrine secretion rate,
epinephrine from infusion (using the Epi pump feature) and
any epinephrine secretion from a possible pheochromocytoma.
The model user may create a pheochromocytoma and
define different levels of severity.
