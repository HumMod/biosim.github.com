CreatininePool.InitialVolDist

Definition

InitialVolDist is initial volume in which creatinine is
distributed. It is a fraction of total body water and is
defined by the parameter VoldDistFract.

InitialVolDist = VoldDistFract * BodyH20.InitialTotalVolume
