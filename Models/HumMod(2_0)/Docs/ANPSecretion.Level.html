ANPSecretion.Level

Definition

ANPSecretion rate can be fixed at a particular level by the
model user. The model user turns the clamp to "on" in the
ANP Secretion display box. The secretion level is then set
using the sliding bar. Secretion rates can be set from zero
pMol/Min up to a maximum of 1000 pMol/Min in increments of 10.
