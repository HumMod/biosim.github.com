AldoSecretion.Rate

Definition

Aldosterone secretion rate is calculated using 6 variables. If the
model user sets the clamp "on", then secretion rate is set by
the"Level" slide bar. Otherwise rate is calculated using Base Rate,
A2Effect, K+Effect and Aldo Tumor secretion. Additionally, since
control of secretion is a function of membrane transport, the effect
of ph, protein, fuel and temperature are important factors. These are
classified as other tissue -function and act as a multiplier effect.
Under normal physiological conditions, they have a value of 1.0.
