A2Pump.Switch

Definition

When the A2pump switch is turned on, the model
user can set infusion rates using the sliding bar
feature in the A2 pump display box.

Physical Units

[Plasma] pG/mL; pMol/L
Plasma Mass nG

Concentration Units

A typical A2 plasma concentration is 10 pG/ml
(recumbent level).
To change from pG to pMol multiply by 0.956
1.046 pG/mL = 1.0 pMol/L
