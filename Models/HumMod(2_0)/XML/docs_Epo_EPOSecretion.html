<name> EPOSecretion - Overview </name>
<label> Overview </label>
<content>

EPO, the hormone that stimulates red blood cell
production, is synthesized in the kidney and liver
in response to hypoxia. Studies in isolated perfused
kidneys suggest that it is tissue oxygenation,
and not arterial PO2 which regulates EPO production
in the kidney. EPO secretion depends on kidney
function. Chronic renal disease can lead
to anemia because of a reduction in EPO secretion.

<name> Base - Definition </name>
<label> Definition </label>
<content>

Base secretion rate is the initial starting secretion rate
of EPO. Base values can be set by the model user.

</content>
</topicdoc>

<topicdoc>
<name> Rate - Definition </name>
<label> Definition </label>
<content>

Rate is defined as base secretion *PO2Effect. Kidney
nephron count and function can affect the rate of
secretion value.

</content>
</topicdoc>

<topicdoc>
<name> PO2Effect - Overview </name>
<label> Overview </label>
<content>

PO2Effect describes the relationship between kidney
tubule PO2 concentration and EPO secretion. Renal tissue
PO2 values are highly variable but a PO2 of 30-40mmHg is
most frequently seen the renal cortex.

Leichtweiss HP, Lubbers DW, Weiss C, Baumgartl H, Reschke W.
The oxygen supply of the rat kidney: measurement of intrarenal
pO2. Pflugers Arch 309: 328 - 349, 1969

</content>
