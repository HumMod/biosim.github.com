<topicdoc>
<name> LeptinPump - Overview </name>
<label> Overview </label>
<content>

The LeptinPump enables the model user to change
plasma concentration of leptin by infusing the peptide
at different rates. Data is taken from a variety of human
and animal studies relating infusion rates to plasma
concentrations of leptin.

Mice Data
Halaas JL, Boozer C, Blair-West J, Fidahusein N, Denton DA,
Friedman JM. Physiological response to lomg-term peripheral
and central leptin infusion in lean and obese mice.
Proc Natl Acad Sci 94: 8878 - 8883, 1997.

Rat Data
Shek EW, Brands MW, Hall JE. Chronic leptin infusion increases
arterial pressure. Hypertension 31: 409 - 414, 1998.

</content>
</topicdoc>

<topicdoc>
<name> Switch - DefinitionLP2 </name>
<label> Definition </label>
<content>

When the LeptinPump switch is turned on, the model
user can set infusion rates using the sliding bar
feature in the LeptinPump display box.

</content>
</topicdoc>

<topicdoc>
<name> Setting - DefinitionLP1 </name>
<label> Definition </label>
<content>

When the LeptinPump switch is turned on, the model
user can set infusion rates using the sliding bar
feature in the leptin Pump display box. Rates of infusion
may be increased from 0, in increments of 0.1, up to
5 U/Min
