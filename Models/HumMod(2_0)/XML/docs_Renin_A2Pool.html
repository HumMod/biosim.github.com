<name> Log10Conc - DefinitionA2P9 </name>
<label> Definition </label>
<content>

Log10Conc converts [A2(pG/mL)] to a logarithmic
value. If [A2(pG/mL)] is less than 1.0, then Log10Conc
has a value of zero.

</content>
</topicdoc>

<topicdoc>
<name> PG_TO_PMOL - DefinitionA2P10 </name>
<label> Definition </label>
<content>

PG_TO_PMOL is a constant used to convert angiotensin II
concentrations in pG/mL to pMol/L.
Since the Molecular Weight of Angiotensin II is 1046,
1.046 pG/mL = 1.0 pMol/L and
1.0 pG/mL = 0.956 pMol/L

PG_TO_PMOL = 0.956

</content>
</topicdoc>
