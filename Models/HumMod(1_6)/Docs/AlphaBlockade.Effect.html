Alpha Blockade - Overview

The model user can block the effect of alpha receptor stimulation.
Receptor blockade is applied at individual organs. Effect is
1.0 when there is no blockade. For example,a 16% blockade results
in an effect of 0.84.
There are a number of drugs known as alpha antagonists which can
be used clinically to cause alpha blockade. These include
phentolamine, prazosin and yohimbine.
