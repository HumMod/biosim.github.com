Viscosity.Clamp

Overview

The clamp feature enables the model user to fix a particular
variable at a set value. In the case of viscosity,
if the clamp is set to TRUE then viscosity would stay fixed at
the initial level. If the clamp is set to FALSE, then viscosity
will change as defined by the relationship of hematocrit to
viscosity.

Physical Units

The viscosity of normal blood is about 3 times as great
as the viscosity of water.
